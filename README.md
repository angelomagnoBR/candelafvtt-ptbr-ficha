# Candela Obscura PT-BR - Ficha Clássica

Módulo para Foundry VTT v13 com:

- tradução PT-BR da ficha do sistema `candelafvtt`;
- tema visual Candela Clássico;
- tamanho fixo da ficha de personagem em 900x740 para manter o layout no lugar;
- remoção do controle de redimensionamento da janela da ficha.
